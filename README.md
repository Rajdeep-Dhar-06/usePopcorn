This project is a movie exploration and watchlist application built using **React** and **Vite**.  
It allows users to search movies via the OMDb API, view detailed information, rate movies, and manage a personal watched list.

The project focuses on practicing modern React concepts such as hooks, side effects, conditional rendering, and reusable components.

---

## ðŸš€ Features

- Search movies using the OMDb API
- View detailed movie information (plot, cast, genre, runtime)
- Rate movies using a custom StarRating component
- Add and remove movies from a watched list
- Watchlist summary with:
  - Average IMDb rating
  - Average user rating
  - Average runtime
- Keyboard support (press `Esc` to close movie details)
- Safe API handling using AbortController
